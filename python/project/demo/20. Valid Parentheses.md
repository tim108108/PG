# [20. Valid Parentheses](https://leetcode.com/problems/valid-parentheses/)
## python
### 解
* 解法：使用stack存右半邊，如果遇到相對應的左半邊pop出來，如果沒有push進去stack，最後stack為回傳True
    ```python
    def isValid(self, s):
        tmp = []
        for i in s:
            if not tmp :
                tmp.append(i)
            elif (i == ")" and tmp[-1] == "(") or (i == "]" and tmp[-1] == "[") or (i == "}" and tmp[-1] == "{"):
                tmp.pop()
            else:
                tmp.append(i)
        if not tmp:
            return True
        else :
            return False
    ```
