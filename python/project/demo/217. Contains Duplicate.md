# [217. Contains Duplicate](https://leetcode.com/problems/contains-duplicate/)
## python
### hash table
* 解法：排序後找出現頻率最高元素是否重複
    ```python
    def containsDuplicate(self, nums):
    tmp = {}
    for i in nums:
        tmp[i] = tmp.get(i,0)+1
    print(tmp)
    if max(tmp.values()) > 1 :
        return True
    else :
        return False
    ```
