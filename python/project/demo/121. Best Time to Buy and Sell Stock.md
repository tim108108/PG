### 解
* 解法：隨時紀錄當天價格，並比較是否比之前買入的價格低，若是有價格更低的天數，便隨時將買入的天數定為價格低的那天。每當遇到賣出價格高的日子，則隨時將賣出的利潤價格與歷史最高利潤價格做比較。
    ```python
    def maxProfit(self, prices):
    profit = 0
    buy = 10000
    for price in prices:
        if (price < buy): 
            buy = price
        if (profit < price - buy): 
            profit = price - buy
    return profit
    ```
