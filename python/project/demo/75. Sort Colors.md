# [75. Sort Colors](https://leetcode.com/problems/sort-colors/)
## python
### 解
* 解法：使用雙指針，從頭到與tail相遇，如果是0移到head，如果是1指針移動，如果是2移動到tail
    ```python
    def sortColors(self, nums):
        head = 0
        tail = len(nums)-1
        i = 0
        while i <= tail:
            if nums[i] ==0:
                nums[head], nums[i] = nums[i], nums[head]
                head+=1
                i+=1
            elif nums[i]==1:
                i+=1
            elif nums[i]==2:
                nums[tail], nums[i]=nums[i], nums[tail]
                tail-=1
        return nums
    ```
