# [844. Backspace String Compare](https://leetcode.com/problems/backspace-string-compare/description/)
## python
### stack
* 解法：用一個stack存暫存，最後進行比較
    ```python
    def backspaceCompare(self, s, t):
        def stack_method(str):
            stack = []
            for char in str:
                if char != '#':
                    stack.append(char)
                elif (char == '#') and (stack != []):
                    stack.pop()
            return ''.join(stack)
        return stack_method(s) == stack_method(t)
    ```
